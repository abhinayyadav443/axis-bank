Update mobile number

  1. Home
  2. Bank accounts
  3. Burgundy
  4. Update details
  5. Update mobile number

Visit the nearest Axis Bank ATM or an Axis Bank Branch. For the security of your account, the mobile number can be changed only through these 2 channels.

ATM: Insert the card > Enter PIN > Select Registrations > Select Mobile Number Update > Select Update > Enter Mobile Number you want to update > Re- enter the Mobile Number.

Branch: Visit nearest Axis Bank Branch and submit Customer Request Form. Click here to download the form.
